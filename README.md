## Simple Gradio Orange Box App

This is a minimal Gradio app that shows:
- **A centered orange box**
- **Three numbered circles on the left side**
- **Three numbered circles on the right side**

The layout is done with Gradio `Blocks` and a bit of custom CSS.

### Setup

From `/Users/matthias/Documents/Experiments/introML`:

```bash
python -m venv .venv
source .venv/bin/activate  # on macOS / Linux
pip install -r requirements.txt
```

### Run the app

```bash
python app.py
```

Then open the local URL that Gradio prints in your terminal.
